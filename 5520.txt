	# Pulse width accuracy depends upon the pulse width and period.
	#
	#   Pulse Period  Pulse Width   Pulse Width Accuracy
	#   ------------  ------------  --------------------
	#   200ns to 2us  2ns to 500ns  unspec'd
	#   2us to 22ms   2ns to 4.9ns  unspec'd
	#   2us to 22ms   4ns to 500ns  5% + 2ns
	#
	# For a pulse width test, the pulse width is in the 5520 FSC
	# Nominal field and the pulse period is in the M5520 FSC Nominal
	# field.  However, MET/CAL bases the 5520A accuracy "lookup"
	# on the 5520 FSC Nominal and MOD1 fields so the pulse period
	# is not known.  Therefore the procedure writer must use the ACC FSC
	# to establish the system accuracy for a pulse width test.
